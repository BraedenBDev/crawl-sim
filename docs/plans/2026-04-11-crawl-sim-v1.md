# crawl-sim v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a working Claude Code skill + shell scripts that audits any URL from the perspective of Googlebot, GPTBot, ClaudeBot, and PerplexityBot — producing a scored report with grades.

**Architecture:** Orchestrator skill (SKILL.md) invokes standalone shell scripts per bot profile. Scripts use curl + jq only (no Node/Python for core). Each script outputs JSON to stdout. A scoring script aggregates results. The skill interprets everything and generates three output layers (score card, narrative, JSON).

**Tech Stack:** Bash, curl, jq, optional Playwright for Googlebot JS render diff. Node.js only for the npm installer (`bin/install.js`).

### Task 3: fetch-as-bot.sh

**Files:** Create `scripts/fetch-as-bot.sh`

Core script. Curls a URL with a bot's User-Agent, captures HTTP status, headers, body (base64-encoded), timing, and word count. Outputs JSON to stdout.

- [ ] **Step 1: Write the script**

Input: `<url> <profile.json>`
Output: JSON with fields: `url`, `bot.id`, `bot.name`, `status`, `timing.total`, `timing.ttfb`, `size`, `wordCount`, `headers`, `bodyBase64`

Use curl `-w` for timing, `-D` for headers, `-o` for body. Base64-encode body for safe JSON embedding. Strip HTML tags and count words for `wordCount`.

- [ ] **Step 2: chmod +x and test**

```bash

./scripts/fetch-as-bot.sh https://example.com profiles/gptbot.json | jq '.status, .bot.name, .wordCount'
```

Expected: `200`, `"GPTBot"`, word count > 0.

### Task 10: compute-score.sh

**Files:** Create `scripts/compute-score.sh`

Input: `<results-dir>` containing all JSON outputs from previous scripts. Computes per-bot scores (0-100) across 5 categories, per-category averages, and weighted overall composite. Outputs JSON with grades.

Scoring weights from spec:
- Accessibility: 25
- Content Visibility: 30
- Structured Data: 20
- Technical Signals: 15
- AI Readiness: 10

Overall weights: Googlebot 40%, GPTBot 20%, ClaudeBot 20%, PerplexityBot 20%.

Grade thresholds: 93+=A, 90-92=A-, 87-89=B+, 83-86=B, 80-82=B-, 77-79=C+, 73-76=C, 70-72=C-, 67-69=D+, 63-66=D, 60-62=D-, <60=F.

- [ ] **Step 1: Write the script** — read all JSON files, compute category scores per bot, compute weighted totals, assign grades
